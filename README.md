# Frontend Starter Kit

**TypeScript, Babel, Webpack, React, Mobx, AntD 4**

## Getting started

-   Clone the repository

```
git clone <this-repo> <project_name>
```

-   Install dependencies

```
cd <project_name>
npm install
npm run dev
```

App accessible at http://localhost:3000
